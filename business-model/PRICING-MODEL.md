# Pricing Model — Handyman & Home Services

## Revenue Model Overview

```
Revenue Streams:
├── 1. Per-Job Revenue (60-70% of income)
│   ├── Flat-rate jobs (preferred)
│   └── Hourly jobs (backup)
├── 2. Recurring Revenue (20-30% of income)
│   ├── Property manager retainers
│   ├── Seasonal maintenance subscriptions
│   └── HOA maintenance contracts
└── 3. Referral/Upsell Revenue (5-10%)
    ├── Subcontracted specialty work (licensed trades)
    └── Material procurement markup
```

## Recurring Revenue Models

### Property Manager Retainer
```
Tier 1: 1-10 units    → $500/month + per-job for 2+ hour tasks
Tier 2: 11-25 units   → $1000/month + per-job for 3+ hour tasks
Tier 3: 26-50 units   → $1500/month + per-job for 4+ hour tasks
Tier 4: 50+ units     → Custom pricing

Includes:
- Priority scheduling (same-day for emergencies)
- Discounted hourly rate ($55-75/hr vs standard $65-95)
- Monthly property inspection (optional add-on)
- Quarterly maintenance visits
```

### Seasonal Maintenance Subscription
```
Basic: $149/quarter ($596/year)
- Gutter cleaning (2x/year)
- HVAC filter change (4x/year)
- Smoke detector battery check (2x/year)

Standard: $249/quarter ($996/year)
- Everything in Basic
- Pressure washing (1x/year)
- Caulking touch-up (1x/year)
- Exterior inspection (2x/year)

Premium: $399/quarter ($1,596/year)
- Everything in Standard
- Deck/fence staining (1x/year)
- Interior paint touch-ups (2x/year)
- Priority scheduling year-round
```

### HOA Maintenance Contract
```
Pricing: Per unit/home in HOA
$15-25/unit/month for common area maintenance
Minimum 20 units, 12-month contract
Includes: Pressure washing, painting touch-ups, minor repairs, seasonal prep
```

## Estimating Framework

### Quick Estimate Formula
```
Flat Rate = (Estimated Hours × Hourly Rate) + Materials + (Materials × 0.20 markup) + Complexity Buffer

Complexity Buffer:
- Simple/routine: +0%
- Moderate (some unknowns): +15%
- Complex (old home, access issues): +25-30%
```

### Always Include in Quotes
1. Scope of work (specific deliverables)
2. Materials included vs. customer-supplied
3. Timeline (start and completion)
4. Payment terms
5. What's NOT included (prevents scope creep)
6. Warranty/guarantee on workmanship
